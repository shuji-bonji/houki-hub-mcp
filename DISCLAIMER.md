# 免責事項・利用範囲

本ソフトウェア（`@shuji-bonji/houki-egov-mcp`、以下「本MCP」）は、日本の法令・通達・裁決・判決等の一次情報へのアクセスを支援する**個人利用向けのセカンドオピニオン・ツール**として設計されています。

## カバレッジ範囲（射程定義）

本MCPは「**法規（法令・通達等）を中核**」とし、関連する一次法情報の**参照ハブ（hub）** として機能します。「houki」が中核、「hub」が周辺の一次法情報を集約することを意味します。

| 種別                                       | カバー方法     | 実装フェーズ                                                                                                  |
| ------------------------------------------ | -------------- | ------------------------------------------------------------------------------------------------------------- |
| 法令（法律・政令・省令・規則）             | コア層         | Phase 1                                                                                                       |
| 行政通達（国税庁・厚労省・JAISH 等）       | 拡張パッケージ | Phase 3 — `@houki-egov/ext-nta` / `ext-mhlw` / `ext-jaish`                                                     |
| 行政裁決（国税不服審判所 等）              | 拡張パッケージ | Phase 3 — `@houki-egov/ext-saiketsu`                                                                           |
| 民事判決（裁判所サイト公開分）             | 拡張パッケージ | `@houki-egov/ext-court` **Stage A**（既存スクレイピング）                                                      |
| 民事判決（年間20万件・オープンデータAPI）  | 拡張パッケージ | `@houki-egov/ext-court` **Stage B**（**2026年度 API 提供開始予定。仕様公開待ちの将来実装** — 進捗注視）        |
| 民事判決（ローカル bulk キャッシュ）       | 拡張パッケージ | `@houki-egov/ext-court` **Stage C**（将来構想）                                                                |
| 条例（地方自治体）                         | 拡張パッケージ | 未定                                                                                                          |
| 金融庁監督指針 等                          | 拡張パッケージ | 未定 — `@houki-egov/ext-fsa` 等                                                                                |

判決を含むのは、**著作権法第13条**で「裁判所の判決、決定、命令」も「権利の目的とならない」公共情報と明示されているため、技術的・法的にハブが扱う対象に含まれます。

> 民事判決のオープンデータ化（年間約20万件公開予定）は、日弁連法務研究財団・最高裁等が進めており、2026年度から運用開始予定です。`@houki-egov/ext-court` Stage B はこの API 仕様公開を待って実装します。

## 本MCPが返すもの

- 条文・通達・裁決事例などの**事実情報（根拠テキストと出典URL）**

LLM はこの事実情報をもとに、分析・説明・論点整理を行うことができます。それはこのツールの通常の使い方です。

ただし、その結果を**業務上の「正解」として扱うことは本MCPの想定外**です。最終判断は利用者自身、または有資格の専門家に委ねてください。

## 3層の責任分離

```
┌─────────────────────────────────────┐
│ 利用者   最終判断と責任                │
│  ↑                                  │
│ LLM      事実からの分析・論点整理       │
│  ↑                                  │
│ 本MCP    一次情報の取得・提示のみ       │
└─────────────────────────────────────┘
```

本MCPは最下層の「事実提供」に徹します。中間層の分析は LLM の能力に依拠し、最終判断は利用者（または有資格者）が行います。

## 想定される使い方

- **プロダクト開発時の法令調査**：個人情報保護法・特商法・景表法・著作権法・資金決済法・電気通信事業法・薬機法・道交法・建基法など、自社プロダクトに関連する法令の初期調査
- **フリーランス・個人事業主の実務**：消費税区分判定・インボイス・青色申告・社会保険加入要件の自己調査
- **スタートアップの業法確認**：新規事業に必要な業法・許認可の洗い出し
- **利用規約・プライバシーポリシーの叩き台**：社内法務・顧問弁護士に相談する前の論点整理
- 個人が自身の案件を調べる際の一次情報への横断アクセス
- AI アシスタントへの「叩き台・論点整理」用の根拠収集
- 法令の学習・自習
- 既存の判断（士業・書籍・記事）に対するセカンドオピニオン的な裏取り

## 想定されていない使い方

- **弁護士法第72条**に定める法律事務の提供（有償・個別具体の法律事務）
- **税理士法第52条**に定める税理士業務の提供
- **社会保険労務士法第27条**に定める社労士業務の独占分野
- その他、有償かつ個別具体の法律・税務・社会保険事務の **業** としての提供

裁判所・税務署・労働基準監督署は「AI がそう言った」を根拠として受け付けません。業として提供する場面では、必ず有資格者（弁護士・税理士・社会保険労務士等）の監督下でご利用ください。

## 一次情報の正確性について

- 法令・通達・裁決は頻繁に改正・公開・更新されます。本MCPが返す情報は、取得時点の一次情報源（e-Gov 法令検索、国税庁、厚生労働省等）の状態を反映します
- 本MCPは一次情報源の誤り・遅延・停止・仕様変更について責任を負いません
- 取得した情報は**取得時点の日付と出典URL**を添えて利用することを推奨します

## 参考: なぜこの立ち位置なのか

- 日本の士業制度（弁護士法72条・税理士法52条・社労士法27条）は「有償」×「個別具体」の二要素が揃った場合に業務独占が発動します
- 個人が自分の案件に AI を使うこと自体は業法の規制外ですが、裁判所や税務署は「AI の回答」を根拠として扱いません
- 本MCPは、個人利用者が AI と協働しつつ**一次情報へのアクセスを簡単にする**ことで、「叩き台の精度」と「有資格者への相談コスト低減」の両方に寄与することを目的とします
- 権威（デジタル庁公式MCP・ベンダー製監修ナレッジ）ではなく、**セカンドオピニオン層**としての網羅性・横断性を提供します

## 3語の守備範囲

|               |            | 法律 | 政令 | 省令 | 規則 | 条例 | 通達 | 裁決 | 判例 |
| ------------- | ---------- | ---- | ---- | ---- | ---- | ---- | ---- | ---- | ---- |
| hourei (法令) | 狭義・公式 | ●    | ●    | ●    | ●    | －   | －   | －   | －   |
| houki (法規)  | 広義       | ●    | ●    | ●    | ●    | ●    | ●    | －   | －   |
| law (英語)    | 最広義     | ●    | ●    | ●    | ●    | ●    | ●    | ●    | ●    |

- **法令 (hourei)**: e-Gov の公式用語。**法律＋政令＋省令＋規則のみ**。通達・判例は含まない（行政内部文書・裁判所判断であり法令ではない）
- **法規 (houki)**: より広い。**通達や条例まで含む**行政一般ルール。ただし判例は微妙
- **law (英語)**: 最広義。**判例も含むあらゆる法的規範**を指せる
